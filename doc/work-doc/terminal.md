## Linux 命令

### Linux系统的文件结构

```
/bin   二进制文件，系统常规命令
/etc  大多数配置文件
/sbin 系统管理命令


```

### 常用命令

|命令|含义|
|----------|--------|
|man mkdir |介绍命令，按 ‘q’退出|
|cd - | 切换到上次访问的目录|
|cd / | 切换到更目录 |
|cd ..  |切换到上一级目录|
|ls     | 查看当前目录下的所有目录和文件|
|ls -a  | 查看当前目录下的所有目录和文件,包括隐藏的文件|
|mkdir test | 在当前目录下创建名为test 的目录|
|rm 文件名| 删除当前目录下的文件|
|rm -f ||
