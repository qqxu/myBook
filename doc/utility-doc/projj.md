### projj

yarn global add projj

projj init
//  set  base direction 用来设置下载的代码存放的位置

从远程拉取 新的 repository
projj add git@gitlab.caijj.net:yanfaerbu/qianduan/creditweb.git

导入 现有的 repository，路径 ~/code
projj import ~/Desktop/XQQ/Project

找 repository 的本地位置
projj find creditweb
